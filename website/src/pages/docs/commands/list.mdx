## Usage

```bash
mx list
```

Lists every documentation package installed in `~/.mandex/cache/` with its version and size.

## Example

```
$ mx list

  Package            Version    Size
  ─────────────────────────────────────
  fastapi            0.115.0    2.1 MB
  react              19.0.0     4.3 MB
  next               15.1.0     5.7 MB
  zod                3.24.0     0.8 MB

  4 packages, 12.9 MB total
```
