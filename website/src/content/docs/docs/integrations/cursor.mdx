## How it works

mandex installs rules to `~/.cursor/rules` that instruct Cursor to use `mx search` for documentation lookups.

## Setup

Automatically detected and installed during `mx init`:

```bash
mx init
```

## What it does

The installed rules tell Cursor to run `mx search` when it needs library documentation, giving it access to accurate, version-pinned docs without relying on web lookups or training data.
